1. Google Cloud Platform 검색 후 접속
2. 좌측 메뉴 클릭 후 compute Engine -> VM인스턴스 클릭
3. 
   - 이름 설정.
   - 라벨 : 인스턴스가 어떤 성격의 인스턴스인지 구분해주는 값
   - 리전 : 인스턴스가 위치할 특정한 지리적 위치
   - 영역 : 리전 안의 여러개의 영역이 존재함.(한쪽 영역에 문제가 발생해도 다른 영역은 문제없이 서비스 가능.)
   - 머신구성 : 설정에 따른 가격 변동.
   - 기밀VM서비스 :
   - 컨테이너 : 
   - 부팅디스크 : (중요) 변경 클릭 후 운영체제 CentOS 그 외 Default 
   - 방화벽 : 추후 설정이 가능하나 번거롭기 때문에 HTTP, HTTPS트래픽 허용. (HTTP : 80번포트 , HTTPS : 443번 포트가 열림)

4. 생성 후 외부 SSH클릭 시 쉘로 접속할 수 있다.(GCP는 웹에서 쉘 접속 가능)
5. ifconfig (엔터) -> 생성된 인스턴스 정보 확인
