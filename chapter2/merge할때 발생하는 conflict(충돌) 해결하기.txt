충돌이 언제, 왜 발생하는가? 
서로 다른 branch를 merge하려고 할 때 발생하고 같은 파일을 각각 다르게 수정했기 때문에 발생한다.

충돌나는 2개의 branch를 merge할 경우 같은 file의 서로다른 로직으로 인해 file에서 merge하려는 두개의 로직을 보여준 뒤 선택하게 처리한다.

충돌 시 file에 문구생김 (VSCode 기준)
 - Accept Current Change : 위의 코드로 변경한다.
 - Accecpt Incoming Change : 아래 코드로 변경한다.
 - Accept Both Change : 두개의 코드를 모두 사용한다.
 - Compare Changes : 두개의 코드를 비교한다.

충돌이 발생하지 않게 하는 법 :
서로 같은 파일을 수정하지 않는다. (코드가 기능별로 세분화 되어있어야 한다.)
