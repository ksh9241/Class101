VM 과 PM의 차이점 :
VM(virtual machine) : 물리적인 하드웨어 시스템에 구축되어 자체 CPU, 메모리, 네트워크 인터페이스 및 스토리지를 갖추고 가상 컴퓨터 시스템으로 작동하는 가상 환경이다.  *하이퍼바이저* 라 불리는 소프트웨어는 하드웨어에서 가상 머신의 리소스를 분석하고 적절히 프로비저닝하여 VM에 사용할 수 있도록 합니다.

PM(physical machine) : 물리적인 컴퓨터 위에 운영체제가 하나 설치되어 있는 것.


CPU Bound Application 과 I/O Bound Application : cpu에서 더 많은 처리를 할 경우 CPU Bound Application 이고 하드에서 더 많은 처리를 할 경우 I/O Bound Application이다.
